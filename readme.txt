== Description ==
Sets default password protection for new posts and provides a backend settings page.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/default-password-protection/` directory.
2. Activate the plugin in the WordPress admin panel.
3. Configure the default password protection through the settings page.

== Frequently Asked Questions ==

= How to set the default password? =
Enable the default password protection and set the password in the plugin settings page.
